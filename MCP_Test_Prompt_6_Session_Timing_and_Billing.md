# MCP Test Case 6: Session Timing & Reserved Billing

หัวข้อการทดสอบ: **การคิดเงินตามเวลาจอง (Reserved Billing) และการควบคุมเวลาใช้งาน (Session Time-up)**

---

### Scenario A: การคิดเงินตามเวลาที่จองไว้ (Reserved Billing)
*เป้าหมาย: ตรวจสอบว่าระบบคิดเงินตามเวลาที่จองขั้นต่ำ แม้จะเล่นจริงน้อยกว่า 1 ชั่วโมง*

1.  **สร้างสมาชิกใหม่**
    *   ใช้ `create_customer_member` ชื่อ "PremiumUser"
2.  **จองโต๊ะ 2 ชั่วโมง**
    *   ใช้ `make_reservation` สำหรับ "PremiumUser" ที่สาขาใดก็ได้ (เช่น "Siam Branch")
    *   เวลา: 10:00 - 12:00 (รวม 2 ชม.)
3.  **เช็คอินตามเวลาจอง**
    *   ใช้ `check_in_reserved` ด้วย Reservation ID ที่ได้
    *   ระบุ `current_time`: "2026-03-11 10:00"
4.  **เช็คเอาต์ทันที (เล่นจริงแค่ 5 นาที)**
    *   ใช้ `check_out`
    *   ระบุ `end_time`: "2026-03-11 10:05"
5.  **ตรวจสอบผลลัพธ์**
    *   **คาดหวัง:** ยอดเงินต้องเป็น ฿30.00 (15 บาท/ชม. x 2 ชม. x 1 คน)
    *   *หมายเหตุ: แม้เล่นจริงแค่ 5 นาที แต่ระบบต้องคิด 2 ชม. ตามที่จองไว้*

---

### Scenario B: บังคับเช็คเอาต์เมื่อมีคนจองต่อ (Forced Checkout)
*เป้าหมาย: ตรวจสอบว่าระบบล็อคการสั่งอาหาร/ยืมเกม เมื่อหมดเวลาและมีคิวจองถัดไป*

1.  **เตรียมการ (ผู้ใช้ 2 คน)**
    *   "User_Alice" จองโต๊ะ TABLE-00000 เวลา 10:00 - 11:00
    *   "User_Bob" จองโต๊ะ TABLE-00000 เวลา 11:00 - 12:00 (จองต่อทันที)
2.  **Alice เช็คอิน** (Current Time: 10:00)
3.  **ข้ามเวลาไปจนหมดเวลาจอง**
    *   ใช้ `get_active_bill` ของเซสชัน Alice
    *   ระบุเวลาสมมติเป็น "11:05" (เลยเวลาจองมา 5 นาที)
    *   **คาดหวัง:** ต้องเห็น Alert แจ้งว่า `TIME IS UP!`
4.  **ลองสั่งอาหารเพิ่มหลังหมดเวลา**
    *   ใช้ `take_order` สำหรับเซสชันของ Alice
    *   **คาดหวัง:** ระบบต้อง Error แจ้งว่า **"Time up! This table is reserved for the next guest. Please proceed to Checkout."**
5.  **เช็คเอาต์** เพื่อคืนโต๊ะให้ Bob

---

### Scenario C: แจ้งเตือนการต่อเวลา (Extension Prompt)
*เป้าหมาย: ตรวจสอบว่าระบบแจ้งเตือนให้ถามพนักงานเรื่องต่อเวลา หากไม่มีคนจองต่อ*

1.  **จองโต๊ะแบบไม่มีคนจองต่อ**
    *   "User_Charlie" จองโต๊ะเวลา 14:00 - 15:00
    *   (ไม่ต้องสร้างการจองอื่นต่อจากนี้บนโต๊ะเดียวกัน)
2.  **Charlie เช็คอิน** (Current Time: 14:00)
3.  **เลยเวลาจองมา 5 นาที** (Current Time: 15:05)
4.  **ลองยืมบอร์ดเกมเพิ่ม**
    *   ใช้ `borrow_board_game`
    *   **คาดหวัง:** ระบบต้อง Error แจ้งว่า **"Time up! ... Would you like to extend your session? (Please contact staff)"**
